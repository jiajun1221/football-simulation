# Trophy Assets

This folder stores trophy images used by the dynamic trophy celebration page.

Expected runtime filenames:

- `premier-league.png`
- `laliga.png`
- `bundesliga.png`
- `serie-a.png`
- `ligue-1.png`
- `fa-cup.png`
- `league-cup.png`
- `champions-league.png`
- `europa-league.png`
- `conference-league.png`
- `copa-del-rey.png`
- `dfb-pokal.png`
- `coppa-italia.png`
- `coupe-de-france.png`
- `default.png`

Initial sources:

- `premier-league.png` and `default.png`: copied from the existing local `Assets/Leagues/premier-league-trophy.png`.
- `laliga.png`: Wikimedia Commons `File:LaLiga.png`.
- `serie-a.png`: Wikimedia Commons `File:Coppa Campioni d'Italia (Serie A).png`.
- `ligue-1.png`: Wikimedia Commons `File:Ligue 1 Trophy 2024.png`.
- `champions-league.png`: Wikimedia Commons `File:UEFA Champions League Trophy - cropped.JPG`.

The remaining filenames were created with the default trophy fallback because Wikimedia rate-limited the batch download during setup. They can be replaced later with competition-specific PNGs using the same filenames.
